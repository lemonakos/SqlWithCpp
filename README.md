# SqlWithCpp
C++ is a powerful language for database applications, and it can be an excellent tool to use with SQL. In this course, instructor Bill Weinman gets you started on leveraging the power of C++ in SQL, starting with the basics, like connecting to a database, performing simple queries, and reading rows from a table. He also explains how to use prepared statements and bind variables, as well as how to build a wrapper class to streamline the SQL interface. Finally, he shows you how to build a specialized application class so you can create an application using what you learned.
